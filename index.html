What is Project Cypress?

Project Cypress is an N-Tiered Web Service oriented system. It's primary goal is the aquisition of event data from the St. Johns Website in order to aggregate and display this data to students via mobile phone applications. The original focus will be on deploying an Android application. Documents with more detailed information will be posted soon.
